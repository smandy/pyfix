FIX Repository Maintenance Release 2007-11-20

FIX4.0 Summary of Changes

Components.xml

Changed <Category> from "Common" to "Session" as a step in introduction of application version independence and transport independence - separation of session from application

From

<Components>
<ComponentName>StandardHeader</ComponentName>
<ComponentType>Block</ComponentType>
<Category>Common</Category>
<MsgID>1001</MsgID>
</Components>
<Components>
<ComponentName>StandardTrailer</ComponentName>
<ComponentType>Block</ComponentType>
<Category>Common</Category>
<MsgID>1002</MsgID>
</Components>

To

<Components>
<ComponentName>StandardHeader</ComponentName>
<ComponentType>Block</ComponentType>
<Category>Session</Category>
<MsgID>1001</MsgID>
</Components>
<Components>
<ComponentName>StandardTrailer</ComponentName>
<ComponentType>Block</ComponentType>
<Category>Session</Category>
<MsgID>1002</MsgID>
</Components>
